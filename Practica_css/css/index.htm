Visión general de CSS

Estas páginas están destinadas a ser exploradas en la consola de desarrollador de su navegador. También debe instalar el complemento Web Developer de Chris Pedrick para jugar con CSS.

Podemos usar el atributo style= en las etiquetas HTML para aplicar varios estilos al texto y a los bloques de texto.

La "cascada" en CSS significa que la configuración de estilo más cercana (monoespacio) "gana" sobre una configuración de estilo (arial) que está "más alejada".

Podemos poner un borde alrededor de un bloque de texto. Probablemente desee tener a mano una hoja de trucos de CSS o utilizar Google para buscar valores de CSS.

Si bien el atributo de estilo funciona, conduce a HTML demasiado detallado, por lo que es más común escribir CSS ⇒ reglas de estilo para aplicar estilos a muchas etiquetas a la vez.
