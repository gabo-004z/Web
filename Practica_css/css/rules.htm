Un encabezado

Pones un conjunto de reglas en una etiqueta <style> y se aplican a partes de los documentos que coinciden con los selectores.

Después de un tiempo, terminamos con demasiadas reglas CSS para colocarlas en el área <head> de cada documento. Una forma más sencilla de incluir una gran cantidad de reglas CSS es colocarlas en un archivo separado e ⇒ incluir ese archivo en cada una de las páginas.
