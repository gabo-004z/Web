"C" significa "cascada"

Order matters. The CSS rule that is "closest" to the tag wins unless a "further away" tag (i.e. border-color) marks itself as !importante.

Incluso el uso del atributo de estilo justo en la etiqueta se desvanece frente a la etiqueta !importante.

Pero siempre puede tomar el control local usando !importante "más cerca" de la etiqueta.

Use !importante con moderacion. Tendemos a usarlo cuando tenemos una hoja de estilo proporcionada por alguien que estamos usando y no podemos cambiar. A veces se está compensando un problema de marcado y se apuesta a solucionar las cosas sin recurrir a !importante si cabe.
