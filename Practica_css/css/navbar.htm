Barra de navegación simple

Lo anterior es un poco de marcado semántico dado y una barra de navegación suele ser una lista de enlaces con uno que va hacia atrás, una indicación de dónde estamos y hacia dónde ir a continuación.

Nos gusta el marcado simple y elegante y preferimos hacer las cosas bonitas usando CSS y no HTML.
