Caja CSS

Etiquetas de bloque como div ,p, y h1 viven en una "caja". Trabajando de adentro hacia afuera, lo tienes.

  1. El contenido de texto o imagen envuelto según corresponda ocupando espacio horizontal y vertical.
  2. El relleno es el espacio entre el contenido y el borde y tiene el mismo color que el texto.
  3. El borde tiene su propio color y ancho.
  4. El margen es el espacio entre el borde y resto de elementos del documento.
  5. También tenga en cuenta que, si bien los elementos ul y li son elementos secunadarios de la etiqueta p tienen su propiocolor de fonso.

  6. Pero puede tener un párrafo dentro de una lista.

  7. Y un elemento de lista puede tener su propio color de fondo.
