Un encabezado

Al colocar las reglas CSS en un archivo separado, se pueden incluir muchas páginas web diferentes con una sola etiqueta de "enlace", generalmente en el "encabezado" del documento. 

  < Tipo de enlace="text/css" rel="hoja de estilo" href="reglas.css">
