Posicionamiento

(1) De forma predeterminada, los bloques de texto se siguen unos a otros.
(2) Uno tras otro.
(3) Una posición relativa se aleja un bloque de donde normalmente se colocaría.
(4) Y el texto después de eso simplemente se establece. Así que ten algo de cuidado con los parientes.
(5) El texto fijo es relativo a la ventana. Mantiene su posición incluso cuando se cambia el tamaño de la ventana o se desplaza. Un vinculo que nunca desaparece.
(6) El texto absoluto es relativo a su elemento padre. En este caso, eldiv se coloca en relación con la etiqueta body. Mantiene su posición incluso cuando se cambia el tamaño de la ventana o se desplaza.
(7) En cierto sentido, los elementos fijos y absolutos se "arrancan" del flujo de renderizado normal y, como tales, no ocupan espacio vertical. estas etiquetas no dejan ningún relleno ni margen.
