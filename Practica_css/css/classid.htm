Selección y estilo de etiquetas

Para evitar poner demasiado en el atributo de estilo, podemos usar los atributos de clase y id para diseñar un subconjunto de las etiquetas.

una etiqueta de identificacion debe ser única en todo el documento, mientras que la etiqueta de clase se puede usar en muchas etiquetas en todo el documento.

Podemos usar la clase a lo largo de todo el documento.

Es muy bueno usar clases. Se puede utilizar en cualquier etiqueta.

¡una etiqueta puede tener mas de una clase!

Incluso puede usar una cadena de selección jerárquica para diseñar etiquetas.

En general, es bueno mantener su HTML lo más limpio y fácil de entender posible como en este simple ejemplo de ⇒ barra de navegacion.
