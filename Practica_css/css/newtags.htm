Etiquetas para CSS

Antes de CSS, todas las etiquetas tenían algún tipo de estilo predeterminado. ¿Ves el margen predeterminado que tenemos encima y debajo de estas etiquetas de párrafo?

Por defecto, el cuerpo usaba una fuente serif. Los enlaces estaban subrayados y en azul si no se visitaban y en púrpura una vez visitados .. Hace mucho tiempo, en una galaxia lejana, el color de fondo predeterminado del cuerpo era gris/gris. Y también tenemos una prueba en video.

Con CSS queríamos algunas etiquetas que no tuvieran un estilo preexistente. Entonces, la etiqueta span se inventó como la nueva etiqueta "en línea" sin estilo.

Y la etiqueta div es una nueva etiqueta de bloque sin estilo, sin relleno, margen, color de fondo ni nada más. Entonces podría marcar bloques con la etiqueta div y no heredar ningún estilo predeterminado.
Y las etiquetas div también se pueden anidar. Agregar los bordes de 1 píxel ocupa un píxel de espacio.
Puede agregar algo de texto en el div externo.
Wow, realmente no hay margen ni relleno en una etiqueta div sin estilo.

Pasemos a ⇒ más formas de seleccionar texto para darle estilo. Y vuelva a usar una etiqueta p para obtener un pequeño espacio en blanco entre párrafos.
