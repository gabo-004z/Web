CSS and Images

Es divertido hacer flotar una imagen hacia la izquierda o hacia la derecha y ver cómo el texto la envuelve. Es posible que desee agregar un poco de margen para que el texto no se amontone. Un em es la "altura de la fuente actual", por lo que se escala cuando se cambia el tamaño de la fuente. of the font is changed.


A veces necesitamos borrar el "envoltorio colgante" cuando queremos que el siguiente párrafo llegue hasta el margen, en efecto, "borrar el flotante".

Las imágenes pueden estar justo en el medio del texto como un personaje. E incluso podemos hacer que una imagen sea un enlace en el que se puede hacer clic a la página de inicio.
